# nginxfmt
format nginx conf file tools

# Usage

```
go get -u https://github.com/litgh/nginxfmt
```

```
Usage of ./nginxfmt:
  -d string
    	nginx conf dir
  -e string
      nginx conf extension (default ".conf")
  -f string
    	format nginx conf file path
  -i	override origin file
  -m  minify program to reduce its size
```
